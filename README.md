# docker2singularity
```
usage: docker2singularity.py [-h] [-f FILE] [--make-singularity]  
                             [--singularity-bootstrap SINGULARITY_BOOTSTRAP]  
                             [--singularity-from SINGULARITY_FROM]  
                             image_name  
```
